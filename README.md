# Data Processing Tool

Integrált adatfeldolgozó megoldások gyűjteménye szakosított üzleti igényekhez. A program moduláris felépítésű, amely lehetővé teszi az egyedi PDF-, tábla- és adatfeldolgozási feladatok automatizálását.

## Modulok

- **Barcode PDF Masolas** - Vonalkód-alapú dokumentum feldolgozás és másolás
- **Cofanet Help** - Coface adatexportálás és feldolgozás
- **KSH Iparági Értékesítés** - Közös Statisztikai Hivatal adatok feldolgozása
- **Merkantil PDF Feldolgozo** - PDF dokumentumok feldolgozása és megtekintése
- **Mouse Mover** - Automatikus egér mozgató

## Technológia

- **GUI Framework**: PySide6
- **Adat feldolgozás**: pandas, openpyxl
- **PDF manipuláció**: PyPDF2
- **Auto-update**: Beépített verziókezelés és frissítési rendszer

## Indítás

```bash
python main_menu.pyw
```
